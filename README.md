# ai-sandbox

Песочница для обучения Claude API в рамках AI-Expert-Journey.

## Установка

```bash
npm install
```

Создай файл `.env` на основе `.env.example` и вставь свой ключ от https://console.anthropic.com.

## Запуск примеров

```bash
node hello.js       # Первый API-вызов
```

## Структура

- `hello.js` — первый API-вызов (Day 1)
- `.env` — твой API-ключ (НЕ коммитить, лежит в .gitignore)
- `notes.md` — заметки по ходу обучения
